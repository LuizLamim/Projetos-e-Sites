Meus Projetos

Desenvolvimento de Landing Page e Rastreamento de Conversões

Criação de uma Landing Page responsiva e interativa para a comercialização de um curso, desenvolvida com HTML, CSS e JavaScript. O projeto incluiu a configuração estratégica do Meta Pixel e Google Tags (GTM) para monitoramento de tráfego, análise de métricas de conversão e otimização de campanhas de tráfego pago.

Desenvolvimento de Agente de IA e Automação de Vendas

Criação e treinamento de um Agente de Inteligência Artificial voltado para o ecossistema da Hotmart, atuando no suporte e conversão de clientes. O projeto contemplou também o desenvolvimento Front-End de uma Landing Page customizada (HTML, CSS e JavaScript), com integração direta entre o agente inteligente e o fluxo de checkout para venda do produto digital.

Configuração de Infraestrutura Web e Apontamento DNS

Gerenciamento e configuração de infraestrutura web para o lançamento de um curso digital de Recuperação Judicial. O projeto envolveu o registro de domínio na HostGator e a configuração técnica avançada da Zona DNS, realizando os apontamentos de registros tipo A e CNAME para integrar e rotear o tráfego com segurança para a plataforma de hospedagem da Hotmart.
